# LabWebsite
SELab & SDTLab 實驗室網頁

## Structure
* Folder 資料夾
  * css - css 相關檔案  
  * data - 關於實驗室的資料，包含教授、成員、計畫等等 (Json 格式)  
  * file - pdf 文件  
  * fonts - 字形相關  
  * image - 照片相關，包含教授、成員、背景等等  
  * js - JavaScript 檔案  
* File 檔案
  * index.html - 英文版頁面 (靜態網頁版本)
  * index_zh.html - 中文版頁面 (靜態網頁版本)
